# MA_Recommender

A recommendation system for deployment scripts with a current focus on docker-compose.

## Getting Started

### Prerequisites

```
Node.js >= v10
```

### Start

#### Install Packages
```
npm i
```

#### Development
```
npm run dev
```

#### Production
```
npm run build
npm run start
```

#### Environment Variables
- GRAKNURI - (def) 'localhost:48555'
